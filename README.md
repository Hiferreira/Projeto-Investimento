# Projeto de Controle de Investimentos

Este projeto foi desenvolvido como parte do desafio da DIO para criação de uma ferramenta de controle de investimentos utilizando o Excel.

## Descrição
A planilha permite organizar e acompanhar investimentos de forma simples, possibilitando o registro de valores aplicados, rendimentos e controle financeiro pessoal.

## Objetivo
Auxiliar no acompanhamento financeiro e na tomada de decisões relacionadas a investimentos, aplicando os conceitos aprendidos durante o curso.

## Ferramentas Utilizadas
- Microsoft Excel
- GitHub
